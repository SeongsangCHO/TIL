### 선수지식

- array 메소드들에 대한 이해

  - prototype에 `map`메소드가 없으면 유사배열 객체나 반복가능한 객체를 얕게 복사해 새로운 Array 객체로 만들어 순회할 수 있도록 배열형태로 반환해야한다 `spread인 [...]`, `Array.from()`을 사용해주도록 하자.

- NodeList는 Array와 다름

  - NodeList는 JS API가 아닌 Browser API임.
  - `Array.isArray`메서드로 배열인지 확인할 수 있음 이는 유사배열임
