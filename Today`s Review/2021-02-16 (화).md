  # 강제상속

  - 상속되지 않는 값도 inherit로 부모의 값으로 상속받을 수 있음.
  - 

  ```jsx
  <div class="parent">
  	<div class="child></div>
  </div>
  
  .parent{
  	position: absolute;
  }
  
  .child {
  	position: inherit; //강제상속
  }
  ```
